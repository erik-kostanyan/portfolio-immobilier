Trigger Word Detection (AI)

Keyword detection is used by many famous modern technologies like Amazon Alexa and Apple Siri. We want the device to start listening when we say a specific keyword, in our case the keyword is "activate".
In this project I created a dataset from scratch using Fourier Transforms to mix our signal "activate" with some generic speech soundbites.
We use this training set to train a model to detect our signal in various sound files, using TensorFlow and Keras in Python. Because we transformed the sounds to images using Fourier Transforms, we used Convolutional Layers and GRU. We end up creating a model with an accuracy of above 92%.
